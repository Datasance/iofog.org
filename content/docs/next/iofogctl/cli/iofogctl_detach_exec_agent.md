## iofogctl detach exec agent

Detach an Exec Session from an Agent

### Synopsis

Detach an Exec Session from an existing Agent.

```
iofogctl detach exec agent NAME [flags]
```

### Examples

```
iofogctl detach exec agent AgentName
```
