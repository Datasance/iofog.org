## iofogctl attach exec microservice

Attach an Exec Session to a Microservice

### Synopsis

Attach an Exec Session to an existing Microservice.

```
iofogctl attach exec microservice NAME [flags]
```

### Examples

```
iofogctl attach exec microservice AppName/MicroserviceName
```
