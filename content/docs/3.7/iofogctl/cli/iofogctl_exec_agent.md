## iofogctl exec agent

Connect to an Exec Session of an Agent

### Synopsis

Connect to an Exec Session of an Agent to interact with its container.

```
iofogctl exec agent AgentName [flags]
```

### Examples

```
iofogctl exec agent AgentName
```
